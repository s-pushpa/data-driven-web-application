About This Website Project

Introduction

Welcome to this website project. This project is built using HTML, CSS, and Flask Python. It aims to provide the detailed collection of recipes. Initially, the website consists of a few recipes, including beverages, salads, steaks, and pastas. The recipes are presented in detail, providing information on preparation time, cook/bake/blend time, and total time required. Additionally, the calorie consumption for each recipe is provided. Each recipe also includes a comprehensive list of ingredients required and step-by-step instructions for preparation. Furthermore, the website provides a nutrition list for each recipe, indicating the nutritional content in various forms.

Installation

  1. To run the application locally on machine, created an account on Replit (https://replit.com/).
  2. Import the project into your Replit workspace by following these instructions:
    1. Log in to Replit account.
    2. Click on the "Import" button located at the top of the Replit dashboard.
    3. Select the "Import from GitHub" option.
    4. Enter the URL of GitHub repository: The Github repository
    5. Click on the "Import from GitHub" button to initiate the import process.
  3. Once the import is complete, navigate to the project's main folder in the Replit workspace.
  4. Created the "app.py" file.
  5. Run the "app.py" file to start the local server.
  6. After the server starts successfully, open a web browser and enter the following URL: http://localhost:5000

Adding HTML Pages and Rendering with Flask

The website consists of multiple HTML pages that are rendered using Flask, adding or modifying the pages is possible using following steps:

  1. Create a new HTML file for page (e.g., "new_page.html") and add HTML code.
  2. In the "app.py" file, import the necessary Flask modules:
    3. from flask import Flask, render_template
  3. Create a route in the "app.py" file to render the new page:
    4. @app.route('/new_page')
def new_page():
    return render_template('new_page.html')
  4. Save the changes and run the "app.py" file again to update the server.
  5. Now access new page by navigating to http://localhost:5000/new_page in browser.
